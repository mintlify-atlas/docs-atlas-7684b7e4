## Overview

The `Promo` model manages promotional content and campaigns in the Yoneily system. It includes automatic file upload handling, SEO-friendly URL generation, and relationships with users, locales, categories, and files. The model is structurally similar to Gallery but designed specifically for promotional content.

## Custom Methods

### beforeSave()

Automatically generates unique, SEO-friendly URL slugs before saving promotional content:

```php
function beforeSave() {
    $currentUrl = Inflector::slug(strtolower($this->data[$this->name]['texto_promo']));
    $claves = Inflector::slug(strtolower($this->data[$this->name]['claves']));
    
    $result = $this->find('all',array(
        'conditions' => array(
            $this->name .'.url' => $currentUrl.''.$claves,
            $this->name .".".$this->primaryKey." != "=>$this->id
        ),
        'fields' => array($this->name.'.*')
    ));

    if ($result !== false && count($result) > 0) {
        $sameUrls = array();
        foreach($result as $record) {
            $sameUrls[] = $record[$this->name]['url'];
        }
    }

    if (isset($sameUrls) && count($sameUrls) > 0) {
        $currentBegginingUrl = $currentUrl.''.$claves;
        $currentIndex = 1;
        
        while($currentIndex > 0) {
            if (!in_array($currentBegginingUrl . '_' . $currentIndex, $sameUrls)) {
                $currentUrl = $currentBegginingUrl . '_' . $currentIndex;
                $currentIndex = -1;
            }
            $currentIndex++;
        }
    }

    $this->data[$this->name]['url'] = str_replace("_","-",$currentUrl.''.$claves);
    return true;
}
```

  <Expandable title="Function Details">
    <ResponseField name="purpose" type="string">
      Generates unique URL slugs from texto_promo and claves fields
    </ResponseField>
    <ResponseField name="behavior" type="string">

      - Combines texto_promo and claves into single URL
      - Checks database for existing URLs with same slug
      - Appends numeric suffix (_1, _2, etc.) if URL already exists
      - Replaces underscores with hyphens in final URL
      - Ensures URL uniqueness across all promos
    </ResponseField>
    <ResponseField name="returns" type="boolean" default="true">

    </ResponseField>
    <ResponseField name="example" type="string">

    </ResponseField>
