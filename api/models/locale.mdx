## Overview

The `Locale` model represents business locations or establishments in the Yoneily system. It manages detailed business information including company details, manager information, contact data, bank accounts, and multiple file uploads for business and location images.

## Behaviors

### MeioUpload Behavior

Handles automatic file uploads for two separate image fields:

```php
var $actsAs = array(
    'MeioUpload' => array(
        'nombre_file' => array(
            'dir' => 'files{DS}locales/otras',
            'create_directory' => true,
            'thumbnailQuality' => 50,
            'useImageMagick' => true,
            'allowed_mime' => array('image/gif', 'image/jpeg', 'image/pjpeg', 'image/png','audio/mpeg','application/x-shockwave-flash','video/x-flv'),
            'allowed_ext' => array('.jpg','.JPG', '.jpeg','.JPEG', '.png','.PNG', '.gif','.GIF','.mp3','.swf','.flv')
        ),
        'ubicacion_file' => array(
            'dir' => 'files{DS}locales/otras',
            'create_directory' => true,
            'thumbnailQuality' => 50,
            'useImageMagick' => true,
            'allowed_mime' => array('image/gif', 'image/jpeg', 'image/pjpeg', 'image/png','audio/mpeg','application/x-shockwave-flash','video/x-flv'),
            'allowed_ext' => array('.jpg','.JPG', '.jpeg','.JPEG', '.png','.PNG', '.gif','.GIF','.mp3','.swf','.flv')
        )
    )
);
```
