  <Steps>
    <Step title="Install CakePHP and dependencies">
      Ensure you have PHP 5.3+ and a MySQL database. Extract the Yoneily source to your web server directory.
      
      ```bash
      # Set proper permissions
      chmod -R 755 sistema_magdaleno/app/tmp
      chmod -R 755 sistema_magdaleno/app/webroot
      ```
    </Step>
    
    <Step title="Configure database connection">
      Update the database configuration in `app/config/database.php` with your credentials.
      
      ```php
      var $default = array(
        'driver' => 'mysql',
        'persistent' => false,
        'host' => 'localhost',
        'login' => 'your_username',
        'password' => 'your_password',
        'database' => 'venezuela_naciones',
        'prefix' => '',
      );
      ```
    </Step>
    
    <Step title="Import the database schema">
      Import the provided SQL file to set up the database structure and initial data.
      
      ```bash
      mysql -u your_username -p venezuela_naciones < localhost.sql
      ```
    </Step>
    
    <Step title="Access the application">
      Navigate to your application URL and log in with administrator credentials to start managing complaints and reports.
    </Step>
  </Steps>
</div>
